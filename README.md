# 📄 ResumeScore AI

An AI-powered Resume Screening System that evaluates resumes against domain-specific job descriptions using Natural Language Processing (NLP) and Sentence Transformers.

---

## 🚀 Features

- Upload resumes in PDF format
- Extract resume text automatically
- Clean and preprocess resume content
- Compare resumes with domain-specific job descriptions
- Generate AI-based Resume Score
- Semantic matching using Sentence Transformers
- Modern Flask web interface

---

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/ResumeScore-AI.git
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
python app.py
```

Open your browser and visit

```
http://127.0.0.1:5000
```
